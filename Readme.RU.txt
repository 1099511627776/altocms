Alto CMS
========

УСТАНОВКА
---------
1. Скопировать файлы движка в нужный каталог сайта
2. Зайти по адресу http://ваш_сайт_на_ls/install/
3. Следовать инструкциям установщика.

КОНФИГУРАЦИЯ И НАСТРОЙКА ДВИЖКА
-------------------------------
Настройки находятся в файле /config/config.php. Для их изменения желательно переопределять эти настройки в файле
config.local.php, это позволит избежать проблем при последующих обновлениях.

Управление плагинами находится по адресу /admin/plugins/

ВОЗМОЖНОСТИ ПОИСКА ПО САЙТУ
---------------------------
Alto CMS поддерживает полнотекстовый поиск по сайту с использованием поискового движка Sphinx.

Соответственно если нужен поиск на сайте, то необходимо установить и настроить на сервере Sphinx, пример
конфигурационного файла(sphinx.conf) находится в каталоге /install/

Официальный сайт проекта http://altocms.ru
